# NAVER-GO

네이버 지도 AI 여정 만들기 프로토타입 (2팀)

## 프로젝트 소개

네이버 지도 앱 내 **AI 기반 여행 여정 생성 기능** 프로토타입입니다.  
사용자가 자연어로 여행 의도를 입력하면, AI가 맞춤형 여행 일정을 생성하고 지도 위에 시각화해주는 경험을 설계했습니다.

### 주요 기능
- AI 대화형 여정 생성 (프롬프트 기반)
- 제주도 여행 일정 타임라인 시각화
- 장소 카드 (베릴, 협재, 온다정, 파르나스, 신창 등)
- 교통수단 연결 및 이동 시간 표시
- 코치마크 기반 신규 진입자 가이드
- 여정 저장 기능
- 모바일 반응형 대응

## 실행 방법

각 버전 폴더의 HTML 파일을 브라우저에서 직접 열거나, 로컬 서버로 실행할 수 있습니다.

```bash
# 최신 버전 실행 예시
cd v1.0
# Live Server 또는 브라우저에서 "AI 여정 만들기.dc.html" 열기
```
